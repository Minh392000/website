Hướng dẫn cấu hình dành cho quản trị viên

  • Truy cập Firebase Console, tạo một Project mới.
  • Tạo một cơ sở dữ liệu Cloud Firestore ở chế độ Thử nghiệm (Test Mode) với quy tắc đọc/ghi tự do.
  • Sao chép thông tin cấu hình API Keys của ứng dụng web (App Web Configuration) và dán đè trực tiếp thay thế đối tượng firebaseConfig ở dòng số 129 trong mã nguồn của tệp HTML này.
  • Mở tệp HTML này trên bất kỳ trình duyệt hoặc máy chủ local nào. Các phiên kết nối khác nhau sẽ đồng bộ đồng thời khi bạn nhấn cập nhật!
